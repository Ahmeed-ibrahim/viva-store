Viva Store is a front-end e-commerce web application built using HTML, CSS, and JavaScript.

Features:
- Clean and modern UI
- Product cards and responsive layout
- Shopping cart system with LocalStorage
- Add / Remove / Clear cart functionality
- Total price calculation
- Dark mode toggle
- Smooth animations and hover effects

This project represents my return to programming after a 45-day break and is my first hands-on experience using JavaScript to implement real interactive web features.

Built with passion, consistency, and a focus on improving front-end development skills.
